# DroidLink Web Installer

This repository contains the DroidLink web-based installer used to flash firmware onto supported DroidLink devices via a modern web browser.

It provides a simple, guided installation experience and does not contain firmware source code.

## What this installer does

- Allows users to install DroidLink firmware using Web Serial
- Works in supported browsers (Chrome / Edge)
- Supports installing firmware for:
  - Master Controller
  - Universal Slave
  - Display Module
- Requires no local toolchains or IDEs

## What this repository does not include

- Firmware source code
- Firmware binaries
- Licensing or activation logic

Firmware binaries are distributed separately to licensed users.

## Usage

End users should access the installer through the official DroidLink installer page:

https://droidlink.github.io/DroidLink_Installer/

No interaction with this repository is required for normal use.

## Supported browsers

- Google Chrome
- Microsoft Edge
